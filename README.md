## 🎯 Project Overview

### What is IRREG?

**IRREG: Integrated Console Application Suite** is a comprehensive C# console application developed as our first-year final project in Fundamentals of Programming. It represents our journey from programming beginners to capable developers who can build functional, user-friendly applications.

### The Vision

> *"To create a single console application that demonstrates every programming concept we learned throughout our first year, while providing practical utility and engaging entertainment."*

### Key Statistics

| Category | Metrics |
|----------|---------|
| **Code Size** | ~1,500 lines of C# |
| **Modules** | 6 major functional modules |
| **Games** | 2 complete classic games |
| **Development Time** | 3 months (October - December 2022) |
| **Team Size** | 4 members |
| **Grade Achieved** | A+ (96/100) |

### Core Values

1. **Educational Excellence** - Demonstrates all first-year programming concepts
2. **Practical Utility** - Provides real-world useful tools
3. **Entertainment Value** - Includes engaging classic games
4. **Professional Presentation** - Visually appealing console interface
5. **Collaborative Development** - Built through effective teamwork

## 📜 Project History

### The Beginning (October 2022)

As first-year students, we were given our final project assignment: create a comprehensive console application demonstrating everything we learned. The requirements were challenging:

**Minimum Requirements:**
- Login system with security
- Minimum 3 functional utilities
- At least 1 game
- Professional presentation
- Complete documentation

**Our Response:**
We decided to exceed expectations by creating a suite of 6 utilities + 2 games, with a polished interface and complete documentation.
